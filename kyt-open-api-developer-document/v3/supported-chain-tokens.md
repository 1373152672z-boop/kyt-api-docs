# Supported Chain Tokens

Currently, the V3 version of the OPEN API provides four external risk-screening related endpoints.

***

### Core Interfaces

| Interface Name                        | Interface URL                             | Special Notes                                                                           |
| ------------------------------------- | ----------------------------------------- | --------------------------------------------------------------------------------------- |
| Address Deposit/Withdrawal Risk Alert | `/openapi/v3/risk/rule/address/screening` | —                                                                                       |
| Real-time Transaction Risk Alert      | `/openapi/v3/risk/rule/tx/screening`      | —                                                                                       |
| Wallet Address Risk Screening         | `/openapi/v3/risk/address/screening`      | —                                                                                       |
| Blacklist Address Detection           | `/openapi/v3/risk/address/blacklist`      | This endpoint supports all EVM-compatible mainnets and all tokens under those addresses |
